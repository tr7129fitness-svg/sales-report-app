# 営業結果レポート

無料公開とランニングコストなしを前提にした、GitHub Pages + Firebase Firestore のWebアプリです。

## 使い方

ホームでは社員別のカレンダー確認と検索ができ、`報告入力` ボタンから入力画面へ移動できます。

Firebase Firestoreに共有保存されるため、同じ公開URLを見る社員同士で同じ報告データを確認できます。

## 現在の入力項目

- 日付
- 営業担当：谷本、會川、上原、ベイ
- 会社名
- 担当者名
- 料金内容：各項目ごとに `なし` または500円単位で選択
- 営業の所感

## 検索

ホームの検索欄では、会社名、担当者名、営業担当、日付、料金内容、営業の所感をまとめて検索できます。社員フィルターと検索は同時に使えます。検索中は該当した報告が箇条書きの一覧でも表示されます。

## 無料運用の注意

無料枠内で運用するため、FirebaseのSparkプランからアップグレードしないでください。
